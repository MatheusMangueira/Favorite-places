# Projeto de Avaliação de Lugares Favoritos

Este projeto foi desenvolvido utilizando a linguagem Java e tem como objetivo coletar informações sobre os lugares favoritos dos usuários, como por exemplo "Salvador, BA". Além disso, permite aos usuários avaliarem esses lugares, calculando posteriormente a média das avaliações para exibir uma lista dos lugares favoritos.

### Funcionalidades Principais

<ul>
  <li> Coleta de Informações: O programa solicita ao usuário que insira o nome do seu lugar favorito. </li>
  <li>Avaliação: Além do nome, o usuário também é solicitado a fornecer uma avaliação para o lugar escolhido.</li>
  <li>Cálculo de Média: O sistema calcula automaticamente a média das avaliações coletadas. </li>
  <li> Exibição de Lista: Ao final da entrada de dados, o programa exibe uma lista dos lugares favoritos com suas respectivas médias de avaliação.</li>
</ul>

### Como Utilizar

<ul>
  <li> Clone o repositório para o seu ambiente local. 
    </br> git clone https://github.com/MatheusMangueira/Favorite-places.git
  </li> 
   
  <li>Abra o projeto em seu ambiente de desenvolvimento Java preferido.</li>
  
  <li>Execute o arquivo principal favoritePlace.java para iniciar o programa. </li>
  
  <li>Siga as instruções apresentadas na interface de linha de comando para inserir seus lugares favoritos e avaliações.</li>
  
   <li>Ao final da entrada de dados, o programa exibirá uma lista dos lugares favoritos junto com suas médias de avaliação.</li>
   
</ul>
